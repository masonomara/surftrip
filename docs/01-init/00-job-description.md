# Frontend Engineer

The Front-End Engineer is responsible for designing and developing user-facing features that deliver intuitive, high-performance web applications. This role focuses on building scalable, responsive interfaces that provide seamless user experiences across modern browsers and devices.

Working closely with product designers, backend engineers, and product managers, the Front-End Engineer transforms design concepts into production-ready applications. The role requires strong expertise in modern web technologies, performance optimization, and user-centered design principles.

The Front-End Engineer plays a key role in shaping the user experience of the company’s products by implementing visually engaging interfaces while ensuring maintainable and scalable code architecture.

## Some of the things you’ll do

- Develop responsive web applications using modern front-end technologies.
- Implement UI/UX designs into high-quality, production-ready code.
- Build reusable components and scalable front-end architecture.
- Optimize application performance, load times, and user experience.
- Collaborate with designers and backend engineers to deliver integrated solutions.
- Maintain and improve existing front-end systems and codebases.
- Ensure cross-browser compatibility and mobile responsiveness.
- Participate in code reviews and maintain coding best practices.
- Implement testing frameworks and maintain code quality standards.

## Requirements

- Bachelor’s degree in Computer Science, Software Engineering, or related field.
- 1-3 years of experience in front-end development.
- Strong proficiency in JavaScript, HTML5, CSS3.
- Experience with modern frameworks such as React, Vue, or Angular.
- Knowledge of responsive design, accessibility, and performance optimization.
- Experience with version control systems such as Git.
